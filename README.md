# Spotiplay
A spotify display with RGB Backlight, 2.8" Touchscreen, and media keys.

## Pins:

Connect these pins to your components.

| Function   | GPIO |
| ---------- | ---- |
| TFT CS     | 10   |
| TFT DC     | 9    |
| TFT RST    | 8    |
| TFT MOSI   | 11   |
| TFT SCLK   | 12   |
| LED Strip  | 18   |
| Vol +      | 1    |
| Vol -      | 2    |
| Rewind     | 3    |
| Play/Pause | 4    |
| Next       | 5    |
